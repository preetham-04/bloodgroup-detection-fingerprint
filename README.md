# Blood Group Detection from Fingerprint

This project predicts human blood groups using fingerprint images and deep learning.

## Tech Stack
- Python
- PyTorch
- Streamlit
- CNN (ResNet9)

## Features
- Upload fingerprint image
- Deep learning model predicts blood group
- Supports 8 blood types:
A+, A-, B+, B-, AB+, AB-, O+, O-

## How to Run

1. Clone repository
git clone https://github.com/preetham-04/bloodgroup-detection-fingerprint

2. Install requirements
pip install -r requirements.txt

3. Run application
streamlit run app.py
